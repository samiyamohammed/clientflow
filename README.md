# 🌐 ClientFlow – Mini CRM for Freelancers

ClientFlow is a simple and intuitive web-based CRM (Customer Relationship Management) tool designed for freelancers and small business owners to manage clients, projects, and payments in one place.

---

## ✨ Features

* 👤 **Client Management**

  * Add, edit, and delete clients
  * Store contact details and notes

* 📁 **Project Tracking**

  * Assign projects to clients
  * Track project status (Pending, In Progress, Completed)

* 💰 **Payment Monitoring**

  * Record payments
  * Track paid vs unpaid invoices

* 📊 **Dashboard Overview**

  * Total clients
  * Total earnings
  * Pending payments

* ⚡ **Fast & Offline-Friendly**

  * Works seamlessly with local storage (no backend required for MVP)

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **State Management:** React Hooks
* **Storage:** LocalStorage (MVP)
* **Styling:** CSS / Tailwind (optional)

---

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/samiyamohammed/clientflow.git
cd clientflow
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the app

```bash
npm start
```

The app will run on:

```
http://localhost:3000
```
